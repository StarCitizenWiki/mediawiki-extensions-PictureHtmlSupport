# PictureHtmlSupport

`wfLoadExtension( 'PictureHtmlSupport' )`

Add to `composer.local.json`:
```json
{
    "extra": {
        "merge-plugin": {
            "include": [
                "extensions/PictureHtmlSupport/composer.json"
            ]
        }
    }
}
```

Run `composer install` in root
